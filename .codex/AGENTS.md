# AGENTS.md

## Project overview
This project implements a real-time, procedural SDF planet renderer using Python (PyOpenGL + GLFW) and GLSL shaders. 
The renderer supports height-mapped terrain, volumetric atmosphere and clouds, and a free-flight FPS camera for orbital/fly-by visualization. 
The codebase is structured into modules for utilities, rendering logic, shader code, and main application entry.

## Stack
- Python 3.10
- PyOpenGL, GLFW
- GLSL (OpenGL 4.1 core profile)
- numpy
